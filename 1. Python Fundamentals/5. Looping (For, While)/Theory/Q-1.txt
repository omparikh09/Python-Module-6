Q-1. Introduction to for and while loops.

ans:
    loop is a repetitive execution of block of code. 

    1. For Loop :- 
        This is a sequence control loop. it's work in a sequence executes one by one.

        Example :-
            for i in range(1,6):
                print(i)

    2. While Loop :- 
        Executes a block of code as long as a condition remains True.

        Example :- 
            count = 0
            while count < 5:
                print(count)
                count += 1
